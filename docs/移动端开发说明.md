# Private Reader 移动端开发说明

## 当前定位

- 技术栈：Flutter
- 首发目标：Android 手机与平板
- 桌面端与浏览器端：后续统一沿 Flutter Web/Desktop 路线推进

## 当前能力

- 登录与会话恢复
- 书架展示
- `TXT / EPUB` 统一正文阅读器
- 目录、书签、批注、阅读进度
- 四主题阅读设置
- 本地离线队列与恢复联网后的自动补偿同步

## 当前不做

- PDF 移动端正式阅读器
- 离线书籍下载
- Flutter Web/Desktop 正式发布产物
- 第二套 Web 阅读器重构
