# Private Reader APP 阅读器 UI 设计规范

> 版本：v1.0  
> 适用范围：Mobile（375–767px）与 Pad（768–1024px）双端响应式  
> 设计参考：微信读书  
> 核心理念：纸墨质感，沉浸阅读

---

## 1. 设计原则

### 1.1 视觉基调
- **克制**：避免过度装饰，让内容本身成为视觉中心。UI 控件仅在用户需要时出现。
- **纸质书感**：封面、阅读背景、阴影均参考实体书的材质与光影。
- **深色沉浸**：夜间模式不是简单的反色，而是降低整体亮度、使用暖灰文字，减少蓝光刺激。
- **一脉相承**：与现有 Web 端的暖棕色系（`#7a4a24`）保持品牌关联，但在 APP 端更克制、更扁平。

### 1.2 内容优先
- 每个页面只有一个视觉重心：书架页是封面网格，阅读器页是正文流。
- 减少卡片嵌套。能使用留白与分割线表达层级时，不添加边框或阴影。
- 不使用营销文案。所有文字均为操作导向：「目录」「批注」「字号」「夜间模式」。

### 1.3 触控友好
- 所有可点击区域不小于 44×44px。
- 底部操作栏在 Mobile 端固定于安全区内（`env(safe-area-inset-bottom)`）。
- 支持手势：侧滑返回、阅读器内左右翻页、下拉关闭底部 Sheet。

### 2.2 字体系统

- **UI 字体**：系统默认无衬线字体栈 `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`
- **正文字体**：按用户选择动态切换。默认使用 `"PingFang SC", "Noto Sans SC", sans-serif`
- **可选正文字体**（与微信读书对齐）：系统默认、仓耳今楷、思源宋体、思源黑体、方正宋三、方正悠宋、方正兰亭黑等

|------|--------|-----|

| 标签/角标 | 11px / 500 | 12px / 500 |

### 2.3 间距与圆角

- **基础间距单元**：4px
- **页面边距**：Mobile 16px，Pad 24px
- **卡片/面板圆角**：Mobile 12px，Pad 16px
- **按钮圆角**：999px（全圆角药丸形）
- **封面圆角**：4px（克制，模拟真实书籍）

## 4. 全局组件规范

### 4.1 顶部导航栏（Mobile）
- 高度：48px（不含状态栏）
- 背景：透明，滚动后变为 `--panel` + `backdrop-filter: blur(12px)`
- 左侧：返回箭头（若适用）
- 中间：书名，单行截断
- 右侧：更多（⋮）或关闭（×）
- 在阅读器中，点击正文可 toggle 显隐，动画 200ms ease opacity

### 4.2 底部操作栏（Mobile）
- 高度：56px + safe-area-inset-bottom
- 背景：`--panel` + 顶部 1px `--line` 分割线
- 图标风格：线性图标，2px 描边，24×24px
- 典型按钮：目录、笔记、亮度/主题、设置、进度
- 点击后对应面板从底部或侧边滑出

### 4.3 侧边栏 Drawer
- **Mobile**：全屏宽 85%，从左侧滑入，遮罩层 `--mask`，手势左滑关闭
- **Pad**：固定宽 320px，左侧悬浮，无遮罩，与阅读区并排
- 头部：标题 + 关闭按钮
- 内容：可滚动，统一使用 16px 页面边距

### 4.4 底部 Sheet（设置面板）
- **Mobile**：从底部滑入，圆角 16px（顶部），占屏高 55–75%，支持手势下拉关闭
- **Pad**：右侧 380px 面板，从右滑入，圆角 16px（左侧）
- 头部：面板标题 + 关闭/完成按钮
- 内容区块：用 12px 分组间距 + 分组标题（13px， `--ink-tertiary`）区分

### 4.5 批注浮动工具条
- 触发：文字选中后弹出
- 位置：选中文字上方（若空间不足则下方），水平居中于选中区域
- 样式：深色（`#2A2A2E`）圆角药丸背景，白色图标/文字，投影 `0 4px 16px rgba(0,0,0,0.2)`
- 按钮：高亮、批注、复制、分享（按功能清单可精简为高亮/批注/复制）

### 4.6 按钮规范
- **主按钮**：`--accent` 背景，白色文字，全圆角，高度 44px，内边距 0 24px
- **次要按钮**：`--bg-soft` 背景，`--ink-secondary` 文字，1px `--line` 边框
- **图标按钮**：44×44px 点击区，透明背景，hover 时 `--bg-soft` 填充

## 5. 页面级设计规范

### 5.1 书架页

**布局结构**
```
[状态栏]
[顶部栏]  头像/昵称（左）  编辑（右）
[封面网格]  3/4/5 列自适应
[底部 Tab Bar]  书架 | 发现 | 我（若需要）
```

**细节**
- 封面：宽高比 2:3，圆角 4px，轻微投影 `0 4px 12px rgba(0,0,0,0.08)`
- 书名：封面下方，13px，单行截断，居中
- 作者/进度：可选，14px，`--ink-tertiary`
- 空状态：居中插图 + 「书架空空如也，去添加一本书吧」+ 主按钮
- 长按封面：弹出操作菜单（删除、详情）

**Pad 差异**
- 顶部栏不显示昵称，改为「书架」标题居中
- 封面网格 4–5 列，间距 20px

> 示意图：`app-reader-ui/bookshelf.html`

---

### 5.2 阅读器页（核心）

**布局结构（Mobile）**
```
[状态栏]
[顶部栏]  ← 返回  书名  ⋮更多
[正文区域]  流式布局，左右 16px 内边距
[底部栏]  ←上一章  进度条  下一章→
[底部操作栏]  目录  笔记  主题  设置
```

**布局结构（Pad）**
```
[左侧目录面板 320px]  [正文区域 居中 max-width 680px]  [右侧批注面板 320px]
```
- Pad 默认三栏，用户可关闭两侧面板进入沉浸式阅读

**正文排版**
- 行高：1.8（用户可选 1.6 / 1.8 / 2.0）
- 段落间距：1em
- 首行缩进：2em（中文排版）
- 图片：最大宽度 100%，圆角 4px

**阅读进度**
- Mobile：底部迷你进度条（2px 高，`--accent` 填充）+ 百分比文字
- Pad：顶部或底部显示章节名 + 百分比

**工具栏显隐**
- 点击正文 toggle 顶部栏 + 底部栏 + 进度条的显隐
- 隐藏后进入全沉浸模式，仅显示正文与系统状态栏

**主题切换**
- 切换时无闪烁，所有颜色通过 CSS 变量即时生效
- 过渡动画：`transition: background-color 200ms ease, color 200ms ease`

> 示意图：`app-reader-ui/reader-light.html`、`reader-night.html`、`reader-kraft.html`

### 5.3 设置面板（字体/主题）

**布局结构（底部 Sheet / 右侧面板）**
```
[头部]  阅读设置          完成
[字号区]  A-  [====●====]  A+
[字体区]  系统默认 | 仓耳今楷 | 思源宋体 | ...
[行高区]  紧凑 | 标准 | 宽松
[主题区]  默认白 | 牛皮纸 | 护眼 | 夜间
```

**细节**
- 字号：滑块范围 90% – 150%，步进 2%，两侧 A-/A+ 按钮支持快速调整
- 字体：网格按钮，3 列，选中态为 `--accent` 边框 + 文字变色
- 主题：色块预览，圆形 48px，选中态外圈 2px `--accent` 描边 + 对勾图标
- 行高：分段控制器（Segmented Control），圆角药丸

> 示意图：`app-reader-ui/reader-settings.html`

### 5.5 批注与笔记管理

**批注列表（侧边栏/独立页）**
```
[头部]  笔记              ×
[统计行]  共 12 条笔记  按时间排序
[列表]
  - 引用文字：15px，--ink，最多 3 行截断
```

**批注编辑（Mobile）**
- 点击「批注」后，底部弹出输入面板
- 顶部显示被引用的文字（只读，--ink-tertiary 背景）
- 多行文本输入框，自动聚焦，键盘弹出
- 操作：选择颜色、保存、取消

**批注编辑（Pad）**
- 右侧批注面板直接展开编辑态
- 输入框占据面板下半部分

## 6. 交互与动效规范

### 6.1 转场动画

| 场景 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 书架 → 阅读器 | 推入（从右向左） | 300ms | ease-in-out |
| 阅读器 → 书架 | 推出（从左向右） | 300ms | ease-in-out |
| 底部 Sheet 弹出 | translateY(100%) → 0 | 300ms | cubic-bezier(0.32,0.72,0,1) |
| 底部 Sheet 关闭 | 0 → translateY(100%) | 250ms | ease-in |
| 侧边栏滑出 | translateX(-100%) → 0 | 300ms | ease-out |
| 遮罩层淡入 | opacity 0 → 1 | 200ms | ease |
| 工具栏显隐 | opacity 0 ↔ 1 | 200ms | ease |
| 主题切换 | background/color 过渡 | 200ms | ease |

### 6.2 手势规范

- **侧滑返回**：Mobile 从屏幕左边缘右滑，触发页面返回（书架/阅读器间）
- **阅读器翻页**：左右滑动切换上一页/下一页（TXT/PDF 分页场景）
- **Sheet 下拉关闭**：手指向下拖拽超过阈值（60px）或速度阈值，关闭 Sheet
- **Drawer 左滑关闭**：从左滑入的目录面板，可左滑关闭
- **长按菜单**：书架封面长按弹出操作菜单

### 6.3 反馈

- 按钮按下：scale(0.97)，100ms
- 列表项点击：背景色闪变 `--bg-soft`，150ms
- 操作成功：底部 Toast，高度 40px，圆角 8px，停留 2s 后淡出

## 8. 开发建议

- 使用 CSS 自定义属性（Variables）实现四主题，避免 JS 切换 class 时的闪烁。
- Mobile 端优先使用 `dvh` / `svh` 处理 100vh 问题，确保底部安全区适配。
- 阅读器正文区域使用 `overscroll-behavior: contain` 防止与页面滚动冲突。
- 字体选择面板中的字体文件建议按需加载，避免一次性引入过多中文字体体积。
- Pad 端的三栏布局可使用 CSS Grid：`grid-template-columns: 320px 1fr 320px`，中间列 `max-width: 680px` + `justify-self: center`。
